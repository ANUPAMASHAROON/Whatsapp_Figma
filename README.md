WhatsApp Prototype using Figma:

Description
This repository contains a WhatsApp prototype designed using Figma. The prototype aims to showcase a modern and intuitive interface for a messaging application.

Features
- User authentication
- Chat interface with text and media support
- Group chat functionality
- Emoji support

Design Decisions
- Using a bottom tab bar for navigation to provide easy access to main features.
- Incorporating Material Design principles for a consistent and intuitive user experience.
